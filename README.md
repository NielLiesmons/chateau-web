# Chateau Web

Landing page for Chateau - A community collaboration platform.

## Getting Started

### Prerequisites

- Node.js 18+ and pnpm

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building

```bash
pnpm build
```

### Preview Production Build

```bash
pnpm preview
```

## Deployment to Vercel

This project uses the Vercel adapter and is ready to deploy:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel will automatically detect SvelteKit and deploy

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Tech Stack

- SvelteKit 2
- Svelte 5
- Tailwind CSS 4
- Vercel (deployment)
